## 斩仇跑酷编辑器

本项目基于守望先锋OverWatch游戏地图工坊WorkShop, 是游戏内英雄斩仇Vendetta的自定义游戏模式。编辑器允许玩家更灵活更直观地创建在游戏内地图中属于自己的跑图点位地图, 同时存在弹球、击杀球、HUD和地图文本等可自由编辑的额外内容Addon,。同时有开拓模式Pioneer可供游玩。开拓模式是基于正常模式推出的可供多人游玩且编辑地图的特殊模式, 在开拓者模式下到达点位才可创建新点位, 系统会自动加载开拓过程中使用的技能, 在完成点位创建后自动限制关卡可使用的技能和技巧。

大部分使用[源氏跑酷编辑器](https://github.com/tylovejoy/genji-framework)代码

**Unknown**适配了斩仇跑酷
